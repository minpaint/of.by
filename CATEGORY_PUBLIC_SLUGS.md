# Public Slugs

Публичные URL теперь редактируются в админке:

- `Категории -> public_slug`
- `Материалы -> public_slug`

Если `public_slug` пустой, сайт использует запасное значение.

## Верхние разделы

| Название | slug | public_slug |
| --- | --- | --- |
| Документы | `news` | `dokumenty` |
| Пожарная безопасность | `news-pozharnaya-bezopasnost` | `pozharnaya-bezopasnost` |
| Промбез | `health` | `prombez` |
| Экология | `ekologiya` | `ekologiya` |
| АРМ | `news-grazhdanskaya-oborona` | `attestatsiya-rabochikh-mest` |
| ГО | `grazhdanskaya-oborona` | `grazhdanskaya-oborona` |
| Вопрос-ответ | `chasto-zadavaemye-voprosy-po-okhrane-truda` | `vopros-otvet` |

## Замечание

В базе есть дубли и служебные категории без наполнения. Для них заданы отдельные безопасные адреса, например `dokumenty-arhiv`, `media-arhiv`, `informatsiya-arhiv`.
